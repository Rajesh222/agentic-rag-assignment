# Data Classification Policy

_Document ID: POL-IT-003_

Meridian classifies data into four categories:

## Public
Marketing materials, public case studies. No restrictions on sharing.

## Internal
Day-to-day operational data. Restricted to Meridian employees and authorized contractors.

## Confidential
Client deliverables, financial data, personnel records. Access on need-to-know basis. Encryption required in transit and at rest.

## Restricted
Privileged client information, M&A data, regulatory filings. Access requires explicit data-owner approval. All access is audited.

## Handling Rules
All confidential and restricted data must be stored in Meridian-approved systems. External storage (USB drives, personal cloud accounts) is prohibited for these categories.
