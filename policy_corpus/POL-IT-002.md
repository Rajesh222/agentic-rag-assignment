# Remote Access Security

_Document ID: POL-IT-002_

## VPN
All remote access to Meridian systems must be through the company VPN. Split-tunneling is disabled.

## Multi-Factor Authentication
MFA is mandatory for all remote access. Hardware tokens are provided to employees accessing sensitive systems.

## Public Wi-Fi
Use of public Wi-Fi for work is permitted only via VPN and with MFA.

## Password Requirements
- Minimum 14 characters
- Mix of upper, lower, numeric, special
- Changed every 90 days
- No reuse of last 10 passwords

## Incident Reporting
Report any suspected security incident to IT Security within 1 hour of discovery.
