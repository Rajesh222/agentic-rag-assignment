# UAE Business Travel — Client Site Visits

_Document ID: POL-TRAVEL-003_

Quick reference for common UAE inter-emirate travel scenarios.

## Approvals
- Same-day travel to another emirate: line-manager approval via email sufficient
- Overnight travel to another emirate: formal Travel Request Form required

## Expense Categories
- Transportation: client-attributable project code
- Accommodation: project code if client-billable, otherwise overhead
- Meals: per diem per POL-TRAVEL-002

## Specific to UAE Government Clients
When visiting UAE government client sites, the following additional requirements apply:
1. Carry valid Emirates ID at all times
2. Arrive 30 minutes before the meeting for security clearance
3. For sensitive ministries (list maintained by Operations), prior security clearance is required — allow 5 working days

## Per Diem
See POL-TRAVEL-002. Note that the UAE within-emirate rules apply when traveling inside your home emirate.
