# Client Engagement Rules

_Document ID: POL-CLIENT-001_

## Engagement Letter
Every client engagement requires a signed engagement letter before work commences.

## Scope Changes
All scope changes must be documented in a Change Request and signed by the client before additional work is performed.

## Invoicing
Invoices are issued per the schedule in the engagement letter. Default is monthly in arrears.

## Communication
Formal client communication (deliverables, scope changes, issues) must go through the designated Engagement Manager.

## Quality Review
All deliverables must be peer-reviewed before delivery to the client.

## Client Confidentiality
Client information is classified as Confidential per POL-IT-003 unless the client has explicitly authorized broader use (e.g., case study).
